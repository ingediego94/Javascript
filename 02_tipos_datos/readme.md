# TIPOS DE DATOS

## Tipos de datos primitivos:
Es un espacio en memoria fijo. Se guarda en la memoria STACK. Si buscamos el dato, este se ubica justo en el bloque que se almacena.
- Number
- String
- Boolean
- Undefined 
- Null
